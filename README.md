# AI Job Market Analysis Dashboard (2020–2026)

This project presents an interactive Power BI dashboard analyzing global AI job market trends from 2020 to 2026.

## Key Insights
- Global distribution of AI job openings
- Average salary by job title
- Job openings by experience level
- Remote work distribution
- Skills required for different AI roles
- Job openings by education level

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Features
- Interactive dashboard
- Dynamic filtering using slicers
- Visual insights into salary, skills, and hiring trends

This dashboard helps understand demand for AI roles, required skills, and hiring patterns across different regions.
